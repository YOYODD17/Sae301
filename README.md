# Saé301 - Développer des parcours utilisateur

Utilisation de l'Intelligence Artificielle
Dans une démarche de transparence et d'optimisation, l'IA a été utilisée comme assistant de programmation dans les domaines suivants :

Développement Frontend : Génération de structures de composants et aide au stylisme (CSS/Frameworks) pour garantir une interface moderne et responsive.

Correction et Optimisation : Révision de certaines fonctions complexes pour améliorer la performance, la sécurité et la lisibilité du code.

Débogage : Aide à l'identification et à la résolution de bugs spécifiques durant la phase de développement.
